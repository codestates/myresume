## ✔️ About MyResume
많은 구직자들 사이에서 강렬한 인상을 남기고 싶으신가요? 
**완벽한 이력서가 핵심**입니다.

### MyResume는 개인 이력서를 효율적으로 작성하는데 도움이 되는 편리한 이력서 작성 도구입니다. 

**정보를 입력하고 원하는 템풀릿을 선택하면 이력서가 생성**됩니다.
